# HR Analytics Dashboard 📊

## Project Overview
This project showcases an interactive **HR Analytics Dashboard** developed using **Power BI**. The dashboard provides insights into employee attrition trends, workforce demographics, and key HR metrics to support data-driven decision-making.

## Key Features
- Total Employees, Active Employees, Attrition Count, Attrition Rate, and Average Age KPIs
- Attrition analysis by Department, Salary Slab, Gender, Age Group, Job Role, and Experience
- Interactive dashboards and visualizations
- DAX measures for KPI calculations and analysis

## Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Visualization

## Business Insights
- Identifies employee attrition patterns across the organization
- Highlights departments and job roles with higher turnover
- Supports workforce planning and retention strategies
- Enables data-driven HR decision-making
